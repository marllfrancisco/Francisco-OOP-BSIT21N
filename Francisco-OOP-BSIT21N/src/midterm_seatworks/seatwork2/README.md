## Reflection
Reading the program requirements, I was able to conceptualize the program flow. I wrote the essential components of a Java program, like the main method and the imported packages. However, the nested if part
is where I am struck with confusion. It was tenfold more difficult to read curly braces on paper and visualize them in an IDE. It took me almost 10 minutes to fix the wrong nested if syntax. Upon passing the paper,

I shall not permit the absence of break statement whenever I write switch-case statement again.
